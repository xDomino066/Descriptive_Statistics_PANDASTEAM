## Exercise 3. - GroupBy

### Step 1, 2 & 3. Import libraries, dataset and assign to drinks
```python
import pandas as pd
df = pd.read_csv("[https://raw.githubusercontent.com/justmarkham/DAT8/master/data/drinks.csv](https://raw.githubusercontent.com/justmarkham/DAT8/master/data/drinks.csv)")
drinks = df
drinks.head()
```

### Step 4. Which continent drinks more beer on average?
```python
drinks.groupby('continent')['beer_servings'].mean().sort_values(ascending=False).head(1)
```

### Step 5. For each continent print the statistics for wine consumption.
```python
drinks.groupby('continent')['wine_servings'].describe()
```

### Step 6. Print the mean alcohol consumption per continent for every column
```python
drinks.groupby('continent').mean(numeric_only=True)
```

### Step 7. Print the median alcohol consumption per continent for every column
```python
drinks.groupby('continent').median(numeric_only=True)
```

### Step 8. Print the mean, min and max values for spirit consumption.
```python
drinks.groupby('continent')['spirit_servings'].agg(['mean', 'min', 'max'])
```
